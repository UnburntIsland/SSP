# Butt Bug 8 Direction Move Direction Prompt Pack

Runtime enemy id: `butt_bug`

Reference frames, use all four as identity/style/scale references for every new direction:

- `assets/images/enemies/butt_bug/move_S_0.png`
- `assets/images/enemies/butt_bug/move_S_1.png`
- `assets/images/enemies/butt_bug/move_S_2.png`
- `assets/images/enemies/butt_bug/move_S_3.png`

Global requirements for every prompt:

- Output 4 separate transparent PNG files, not a sprite sheet.
- Keep the exact same canvas size as the reference frames.
- Keep the same monster identity, color palette, pixel density, visual size, visual center, and animation quality.
- Do not add text, UI, scene background, black background, or white background.
- Do not make the bug become another creature.
- Every frame must have clear motion, not just a tiny position shift.
- Movement style: quick skittering crawl with small leg/body changes and a fast bug-like rhythm, not disgusting.
