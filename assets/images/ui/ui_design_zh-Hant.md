# 《森循島：污染潮倖存者》UI 設計規格

本文件定義像素風 UI 的視覺語言與畫面結構。概念圖可作為風格參考，實作時文字應由 HTML / Canvas 真正渲染，避免把 UI 文字做死在圖片中。

概念圖：

- `assets/images/ui/concepts/eco_adventure_ui_mockup_sheet_v1.png`

## UI 核心方向

主題是「永續教育 × 生態冒險」，視覺應像環保行動冒險遊戲，而不是一般奇幻 RPG。

設計關鍵：

- 像素風，但文字必須清楚。
- 木框、葉片、水滴、回收符號、太陽能、土壤紋理作為輕量裝飾。
- 裝飾只放在角落、圖示、分隔線與按鈕狀態，不遮住資訊。
- 按鈕、卡片、HUD 條使用一致的深色外框與淺色內底。
- 自然色為主，污染色只作為警示或敵方/危機元素。

## 像素 UI 規則

- 外框建議 3-4px 深色像素線。
- 面板內距至少 16px，避免文字碰到木框。
- 圓角最多 6-8px，保持像素方正感。
- 按鈕高度至少 44px，HUD 小按鈕至少 32px。
- 文字與圖示不要使用過細線條。
- Canvas 中渲染圖片時關閉平滑縮放：`imageSmoothingEnabled = false`。

## 字體建議

主要 UI 字體：

```css
font-family: "Noto Sans TC", "Microsoft JhengHei", "PingFang TC", sans-serif;
```

若需要更強像素感，可只在標題或按鈕使用 pixel font；長文字仍建議用清楚的繁中字體。

## 1. 開始畫面

### 內容

- 標題：《森循島：污染潮倖存者》
- 按鈕：`開始遊戲`、`角色選擇`、`商店`、`圖鑑`
- 顯示目前持有：`循環幣`
- 背景：海岸與森林意象，帶少量污染被淨化的光點。

### 版面

- 背景使用第一關海岸/森林混合概念，降低對比並模糊細節。
- 標題置中偏上，使用木牌或葉片徽章承載。
- 主按鈕垂直排列於中間或右中區域。
- 循環幣顯示放在右上角，使用青綠圓幣 + 回收符號。

### 按鈕狀態

| 狀態 | 表現 |
| --- | --- |
| Default | 木框 + cream 底 + 深色文字 |
| Hover | 葉綠外發光 1-2px，按鈕微上移 |
| Pressed | 內陰影，文字下移 1px |
| Disabled | 灰化，降低飽和度 |

## 2. 角色選擇畫面

### 內容

三張角色卡：

| 名稱 | 定位 | 初始技能 | 主色 |
| --- | --- | --- | --- |
| 森林巡守員 | 守護者 | 種子葉刃 | 綠 / 棕 |
| 海岸淨灘者 | 行動派 | 回收網 | 藍 / 白 |
| 太陽能工程師 | 科技支援 | 太陽脈衝 | 黃 / 橘 / 灰 |

### 卡片結構

由上到下：

- 頭像區：角色半身或小立繪。
- 名稱：大字。
- 定位：短標籤。
- 初始技能：圖示 + 技能名。
- 被動或短描述：最多 2 行。

### 選取狀態

- 卡片外框變成葉綠或太陽黃。
- 卡片底部顯示小型葉片/水滴/太陽能符號。
- 未選取卡片保持低對比，但不變灰。

## 3. 商店畫面

### 內容

- 顯示目前 `循環幣`
- 升級項目使用卡片式設計。
- 每張卡包含：圖示、名稱、價格、等級。

### 升級卡片範例

| 名稱 | 圖示 | 價格 | 等級 |
| --- | --- | ---: | --- |
| 土壤修復 | 土壤 + 新芽 | 80 | Lv.1 |
| 回收效率 | 回收符號 | 120 | Lv.2 |
| 太陽能蓄電 | 太陽能板 | 160 | Lv.1 |
| 淨水補給 | 水滴 | 100 | Lv.3 |

### 規則

- 可購買：價格用太陽黃或青綠。
- 不足金額：價格轉灰，按鈕禁用。
- 已滿級：顯示 `MAX`，卡片加葉片徽章。

## 4. 遊戲 HUD

HUD 必須優先不遮擋戰鬥視野。

### 頂部 HUD

- 左上：血量條 `HP`
- 中上：時間，例如 `04:32`
- 右上：循環幣數量

### 底部 HUD

- 底部橫向經驗條 `XP`
- 左下或下方：等級 `Lv. 3`
- 右下：當前技能圖示列

### 視覺規則

- 血量條：木框 + coral fill。
- 經驗條：細長 cyan fill，放在最底部。
- 技能圖示：32-44px 方形槽，帶冷卻遮罩。
- HUD 背板使用半透明深色或低飽和木框，不要整片遮住畫面。

## 5. 升級選項卡片

### 內容

升級時畫面暫停，中央顯示三張並排卡片。

每張卡包含：

- 技能圖示
- 名稱
- 效果
- 簡短說明

### 範例文案

| 名稱 | 效果 | 說明 |
| --- | --- | --- |
| 種子葉刃 Lv.2 | 投射物 +1 | 以種子擴散守護森林生命。 |
| 回收網 Lv.2 | 範圍 +20% | 聚攏海廢並提升淨化效率。 |
| 太陽脈衝 Lv.2 | 冷卻 -10% | 將再生能源轉為淨化光波。 |

### 卡片狀態

- Hover：卡片微上移 2px，外框發光。
- Selected：播放葉片或水滴粒子。
- Rarity：不建議使用傳統 RPG 稀有度顏色；改用永續主題分類：
  - 生態：綠
  - 海洋：藍
  - 能源：黃
  - 修復：棕 / 青綠

## 圖示語言

| 功能 | 圖示方向 |
| --- | --- |
| 循環幣 | 圓形青綠硬幣 + 回收箭頭 |
| 生命 | 水滴或葉心，但血量 fill 使用 coral |
| 經驗 | 小葉片 / 種子光點 |
| 商店 | 木箱 + 回收標誌 |
| 圖鑑 | 米白卡片 + 葉片角標 |
| 太陽能 | 小太陽 / 太陽能板 |
| 淨水 | 水滴 / 泉水波紋 |

## UI 尺寸建議

以 960x600 Canvas 為基準：

| 元件 | 建議尺寸 |
| --- | ---: |
| 主按鈕 | 220x52 |
| 角色卡 | 240x340 |
| 商店卡 | 280x96 |
| 技能圖示 | 40x40 |
| HUD 血量條 | 220x24 |
| HUD 經驗條 | 畫面寬度 - 48 x 12 |
| 升級卡 | 250x300 |

行動裝置：

- 角色卡改為橫向滑動或單欄。
- 升級卡片可改成直向三張堆疊。
- HUD 技能列保持右下，但縮成 32px 圖示。

## 實作注意

- 不要把所有 UI 做成大圖；面板、按鈕、圖示、條狀元件應拆成可重用元件。
- 文字全部由程式渲染，便於在升級、商店、圖鑑中更新內容。
- 裝飾性葉片與水滴可做成 16x16 / 24x24 overlay 圖示。
- 回收符號、太陽能板、水滴等圖示應保持同一描邊粗細。
- 污染色只用於危機提示，不作為常態 UI 主色。
