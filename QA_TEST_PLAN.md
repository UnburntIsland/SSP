# QA 測試計畫

專案：`森循島：污染潮倖存者`

本輪目標是驗證現有 2D Survivors-like Roguelite 的啟動、流程、遊玩、升級、商店、存檔與圖鑑，不新增大型功能、不改變核心玩法方向。

## A. 基礎啟動測試

- `index.html` 可以正常載入。
- 沒有 JavaScript console error。
- CSS 正常載入，主選單有樣式。
- Canvas 存在且尺寸有效。
- 主選單顯示遊戲標題與主要按鈕。

## B. 選單流程測試

- 開始畫面 → 角色選擇。
- 開始畫面 → 商店。
- 開始畫面 → 圖鑑。
- 各畫面可返回主選單。
- 循環幣在主選單、商店、HUD 中正確顯示。

## C. 角色選擇測試

- 三個角色都能被選擇。
- 每個角色的名稱、定位、初始技能、被動效果正確顯示。
- 選角後可以進入第一關。
- 不同角色的初始能力差異生效：
  - 森林巡守員：最大生命值較高。
  - 海岸淨灘者：拾取範圍較大。
  - 太陽能工程師：技能冷卻較短。

## D. 第一關遊玩測試

- 玩家可用 WASD / 方向鍵移動。
- 玩家不會移出地圖邊界。
- 敵人會從玩家周圍畫面外生成。
- 敵人會追蹤玩家。
- 武器會自動攻擊。
- 敵人受傷後會扣血或被淨化消失。
- 敵人掉落經驗。
- 玩家可拾取經驗。
- 經驗條會增加。
- 升級畫面會暫停遊戲。
- 選擇技能後可繼續遊戲。

## E. 戰鬥與數值測試

- 玩家接觸敵人會扣血。
- 玩家死亡會進入失敗畫面。
- 存活到關卡時間結束會進入勝利畫面。
- Boss 會在指定時間出現。
- 技能冷卻、傷害、範圍有基本作用。
- 敵人數量提高時沒有嚴重卡頓或錯誤。

## F. 商店與存檔測試

- 結算後獲得循環幣。
- 循環幣存入 `localStorage`。
- 重新整理後循環幣仍存在。
- 商店購買會扣除循環幣。
- 商店升級會保存。
- 商店升級會影響下一局。

## G. 圖鑑測試

- 知識卡可被解鎖。
- 圖鑑可顯示已解鎖內容。
- 圖鑑資料可保存到 `localStorage`。

## 測試模式需求

使用 `index.html?test=1` 啟動：

- 關卡時間縮短為 30 秒。
- 經驗需求降低。
- 循環幣獲得倍率提高。
- 敵人生成可預測。
- 可快速測試勝利、失敗、升級、商店與存檔。
- 正式模式不能受到影響。
